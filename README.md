# AI-Driven Cyber-Resilient IoT Platform for Real-Time Water Leakage Monitoring and Detection System in Smart Cities

## Overview

This project is a prototype dashboard for monitoring water leakage in smart cities using IoT sensors and AI-based analytics. It provides real-time visualization of leakage events, sensor status, pipeline information, and cyber security monitoring.

The dashboard is designed for municipal authorities to monitor underground water distribution networks efficiently and support timely maintenance decisions.

---

## Features

- Real-time water leakage monitoring
- IoT sensor-based data visualization
- Pipeline information management
- Water loss monitoring
- Flow rate monitoring
- Pressure monitoring
- Sensor health status
- Cyber security status
- Smart search functionality
- Google Maps location view
- Responsive dashboard design
